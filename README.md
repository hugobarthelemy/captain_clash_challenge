# Jouer en local

## Installation

```
git clone git@github.com:hugobarthelemy/captain_clash_challenge.git
cd captain_clash_challenge
bundle install
rake db:create
rake db:migrate
rake db:seed
```

## Lancer le jeu
```
bin/rails s

```

## Acceder au jeu

```
http://localhost:3000/
```
